Built to work with Chromium Edge Driver
1. 	Download Chromium Edge here: https://www.microsoft.com/en-us/edge 
	and install
2. 	Download Chromium Edge Driver (left side) suitable for Chromium Edge installed during step 1 here: https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/#downloads 
	and unpack
3. 	Copy Chromium Edge Driver executable unpacked during step 2 and paste under UI > bin > Release
4.  Launch app using executable Credite.exe in UI > bin > Release
